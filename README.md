## Tech Stack

I built this site using some of my favorite tools:

```bash
Frontend: Next.js + React + TypeScript
Styling: Tailwind CSS (because life's too short for vanilla CSS)
Spice: Framer Motion for those smooth animations
Home: Deployed on Vercel
```

## Want to Run This Locally?

1. Grab the code:

```bash
git clone https://github.com/samuelcorsan/disam.dev.git
cd disam.dev
```

2. Get the dependencies:

```bash
npm install
```

3. Fire it up:

```bash
npm run dev
```

## Deployment
