# Samuel's Portfolio

A clean, minimal portfolio showcasing my projects and journey as a developer from Spain.

## 🚀 About

I'm Samuel, a 15-year-old full stack developer from Spain. I've been passionate about coding since I was 9 and love learning something new every day.

## 🛠️ Built With

- **Astro** - Static site generator

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
